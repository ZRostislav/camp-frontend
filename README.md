# QR-вход для участника + сканер

## Что добавлено

1. **Кнопка «Показать QR для входа»** на странице участника
   (`/participants/:id`, видна любому персоналу). Открывает модалку с
   QR-кодом, который ведёт на `/login` с уже подставленными фамилией и
   именем участника — сканирующему остаётся ввести только код доступа.
   Если на телефоне уже открыта чья-то сессия — его сразу перекинет:
   - на `/profile/me`, если это сам участник;
   - на `/participants/:id`, если это персонал.

   Код доступа **не** зашит в ссылку — сама ссылка безопасна для показа
   на экране.

2. **Свой сканер QR** (камера устройства, без сторонних UI-библиотек) —
   добавлен в `layout.component` (кнопка «Сканировать QR» в боковом меню,
   мобильном меню и в шапке участника) «на всякий случай», чтобы можно
   было отсканировать любой QR этого сайта прямо из любого раздела.
   Работает и на iPhone (Safari) — учтены нюансы `playsinline`/`muted`,
   без которых видео на iOS разворачивается на весь экран.
   Есть запасной вариант — ручной ввод ссылки, если камера недоступна.

## Новые npm-зависимости

```
npm install
```

Добавлены в `package.json`:
- `qrcode` — рендер QR-кода в `<img>` (data URL)
- `jsqr` — распознавание QR-кода из кадра видео
- `@types/qrcode` (dev)

## Изменённые файлы

- `package.json` — новые зависимости
- `src/app/shared/icon.component.ts` — добавлены иконки `qr-code`, `scan-line`
- `src/app/services/qr-code.service.ts` **(новый)** — генерация ссылки и QR-картинки
- `src/app/shared/qr-display-modal/*` **(новый)** — модалка показа QR
- `src/app/shared/qr-scanner/*` **(новый)** — модалка сканера камерой
- `src/app/pages/profile/profile.component.ts/.html` — кнопка и модалка QR
  на странице участника
- `src/app/components/login/login.component.ts/.html` — чтение
  query-параметров QR-ссылки (`mode`, `pid`, `ln`, `fn`), подстановка
  фамилии/имени, автопереход, если сессия уже есть
- `src/app/components/layout/layout.component.ts/.html` — кнопка и
  модалка сканера в сайдбаре/топбаре/мобильном меню

## Проверено

без ошибок TypeScript/шаблонов.
